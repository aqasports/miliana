Key points:
- Main Flutter code is in the `lib/` directory (pages, models, services, widgets)
- Android and iOS native folders for platform-specific code
- Firebase config in `lib/firebase_options.dart` and `android/app/google-services.json`
- Web and desktop support present
- Assets (fonts, images) in `assets/`
- Additional web app in `miliana/`
